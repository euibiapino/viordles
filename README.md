# Viordles

Bot do Discord para servidor da comunidade. Gerencie listas de filmes, séries, animes e jogos com votações, avaliações e eventos.

## Stack

- [discord.js](https://discord.js.org/) v14
- PostgreSQL via `pg`
- [TMDB API](https://www.themoviedb.org/documentation/api) para posters de filmes e séries
- [RAWG API](https://rawg.io/apidocs) para imagens de jogos
- PM2 para gerenciamento de processo
- Hospedado na Oracle Cloud

## Setup

```bash
npm install
```

```bash
node src/index.js
```
